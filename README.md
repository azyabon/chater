## Technologies stack
# Frontend
- NextJS
- TypeScript
- Redux/redux-thunk
- styled-components
- socket.io
# Backend
- ExpressJS
- MongoDB
- Mongoose
- ts-node
- socket.io
- JWT
## Getting start
```bash
# Frontend (client folder)
1. yarn install
2. yarn dev
# Backend (server folder)
1. yarn install
2. yarn start
```
